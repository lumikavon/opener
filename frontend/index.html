概览

Opener 是一款离线可用的桌面启动器，支持多种条目类型与快捷键操作。

主要功能

  • 支持 App / URL / 文件 / 目录 / 命令 / WSL / SSH / 脚本 / 快捷方式 / AutoHotkey
  • 模糊搜索与键盘导航
  • 脚本模板与变量替换
  • 快捷键绑定（应用级与全局）
  • 导入/导出配置
  • 敏感信息安全存储，离线运行

新增条目

  1. 打开设置（齿轮按钮）
  2. 进入“条目”标签页
  3. 点击“新增条目”
  4. 选择类型并填写对应字段

键盘操作

  • ↑/↓：在搜索结果中移动
  • Enter：执行选中条目
  • Esc：清空搜索或关闭弹窗

脚本模板

进入“模板”标签页，点击使用按钮，填写变量后即可生成条目。模板变量使用 {{variable_name}} 语法。

导入/导出

在“导入/导出”中可导出全部配置，或从 JSON 文件导入并选择合并策略。

依赖说明

Windows 下使用 AutoHotkey 功能需要安装 AutoHotkey v2。

可通过 winget 安装：winget install AutoHotkey.AutoHotkey
